a simple app to remix content into tweets

## features

1. paste in text to remix
2. click a button to remix into tweets
3. send request to api endpoint
4. see the remix in output box
5. add other features as we go

## tech stack

1. React
2. Tailwind
3. Vercel
4. OpenAI API
